# API-контракт сервиса «Запись на звонок»

Единый источник правды для фронтенда и бэкенда (подход Design First).
Контракт описан на [TypeSpec](https://typespec.io) и компилируется в OpenAPI 3.0.

## Структура

- `main.tsp` — исходное описание контракта (сущности, эндпоинты, ошибки).
- `tspconfig.yaml` — конфигурация компилятора и эмиттера OpenAPI.
- `dist/openapi.yaml` — сгенерированная OpenAPI-спецификация (артефакт, его и потребляют фронт/бэк).

## Команды

```bash
npm install        # установить зависимости TypeSpec
npm run build      # скомпилировать main.tsp -> dist/openapi.yaml
npm run watch      # пересборка при изменениях
```

> Требуется Node.js ≥ 20. Набор пакетов закреплён на ветке `~0.67.x`
> (последняя с поддержкой Node 20). Для Node ≥ 22 можно поднять до `latest`.

## Как менять контракт

1. Правим `main.tsp`.
2. `npm run build` — проверяем, что компилируется без ошибок.
3. Коммитим `main.tsp` **и** обновлённый `dist/openapi.yaml`.
4. Синхронно вносим правки во фронтенд и бэкенд по новому контракту.
